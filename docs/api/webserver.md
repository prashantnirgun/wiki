# Web Server

## Python 2.x

```python
python -m SimpleHTTPServer 8000
```

## Python 3.x

```python
python -m http.server 8000
```

## Node

```bash
npm install -g http-server   # install dependency
http-server -p 8000
```

## PHP

```php
php -S 127.0.0.1:8000
```

## IIS

```bash
"C:\Program Files (x86)\IIS Express\iisexpress.exe" /path:C:\MyWeb /port:8000
```
