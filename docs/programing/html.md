# HTML

## Special Codes

| Character | Entity  | Number  | Description        |
| --------- | ------- | ------- | ------------------ |
| "         | \&#34;  | \&quot; | quotation mark     |
| '         | \&#39;  | \&apos; | apostrophe         |
| &         | \&#38;  | \&amp;  | ampersand          |
| <         | \&#60;  | \&lt;   | less-than          |
| >         | \&#62;  | \&gt;   | greater-than       |
|           | \&#160; | \&nbsp; | non-breaking space |
